# 🚀 AttendX - AI Smart Attendance System

> **Scan. Smile. Mark Attendance.**  
> Disrupting traditional biometric systems using AI + Webcam

---

## 📌 Overview

**AttendX** is an AI-powered smart attendance system that uses a **webcam instead of expensive biometric hardware**.

💡 Traditional biometric systems cost ₹25,000–₹45,000  
🔥 AttendX reduces this cost to just a **₹1,000 webcam**

It uses **Face Recognition + QR Backup System** to provide a **secure, fast, and contactless attendance solution**.

---

## 🎯 Problem

- Expensive biometric machines 💸  
- Limited access (only office PC) 🖥️  
- No backup if face scan fails ❌  
- Maintenance cost is high 🔧  

---

## 💡 Solution

- 🎥 Webcam-based Face Recognition  
- 📱 QR Code backup system  
- 🌐 Cloud-based access (anywhere)  
- 🔒 Secure login (mobile-number based)  
- ⚡ Real-time attendance tracking

## 🏗️ How It Works

1. User registers with mobile number  
2. Face data is captured via webcam  
3. System trains AI model  
4. On arrival:
   - Face is detected via webcam 🎥  
   - Attendance is automatically marked ✅  
5. If face fails:
   - QR code scan backup 📱
